Breast Treatment Planning Checklist

  • Set CT reference point at the start of the plan for accurate shifts to the isocenter.
  • Verify the correct dose protocol.
  • If DIBH is successful, check for gating use.
  • Ensure proper isocenter positioning.
  • Name the plan and course correctly based on treatment site.
  • Ensure proper field arrangements for BCS & CW cases.
  • Verify SCV planning.
  • Check IMC cases.
  • Ensure Boost plan is correctly applied.
  • Select Boost technique based on case.
  • Check wing board cases.
  • Ensure correct setup for bilateral cases.
  • Apply Mixed Energy Technique for large breasts.
  • Apply Mixed Electron Technique for Boost if single energy is insufficient.
  • Add additional fields if necessary.
  • Use wedges if required.
  • Adjust MLC settings.
  • Apply bolus in T4 cases.
  • Match fields with MLC.
  • Handle high dose in soft tissue.
  • Verify treatment protocols & dose limits.
  • Ensure dose constraints are met.
  • Resolve merging issues.
  • Special case considerations.
  • Finalize treatment setup.
  • Ensure correct gating parameters for DIBH cases.
  • Verify boost treatment.
